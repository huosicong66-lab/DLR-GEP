# AQI Prediction Using Hybrid Machine Learning Models

This project investigates air quality index (AQI) prediction using multiple machine learning and evolutionary algorithms, including hybrid models.

## Models
- Linear Regression (LR)
- Genetic Algorithm (GA)
- Genetic Programming (GP)
- Gene Expression Programming (GEP)
- Graph Neural Network (GNN)
- Random Forest (RF)
- Hybrid Models: LR-GA, LR-GP, LR-GEP

## Dataset
Daily air quality data in Beijing (2022–2024), including:
PM2.5, PM10, SO2, NO2, CO, O3, and AQI.

## Results
Model predictions and evaluation metrics are provided in the `results/` directory.

## Usage
Run each script independently to reproduce the results.
